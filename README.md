# ID3 Decision Tree Implementation

This project implements the ID3 (Iterative Dichotomiser 3) algorithm from scratch using Python.

## Features
- Entropy calculation
- Information Gain
- Recursive tree building
- Decision tree visualization using Graphviz

## Dataset
Weather dataset (weather.csv)

## Output
- Decision tree generated and saved as PNG (id3_tree.png)

## Tools Used
- Python
- Graphviz

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the notebook:
   Open id3_decision_tree.ipynb in Jupyter or Google Colab

## Concepts Covered
- Decision Trees
- Entropy
- Information Gain
- ID3 Algorithm
